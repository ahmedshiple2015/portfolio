# Mechatronics & Industrial Automation Portfolio

A premium, high-performance portfolio website for **Abdelrahman Osama**, a Mechatronics and Industrial Automation Engineer. This project features a modern, dark-themed aesthetic with interactive elements and smooth animations.

## 🚀 Key Features

- **Dynamic Hero Section**: Interactive typewriter effect for professional titles.
- **Custom Cursor Glow**: A smooth, radial light effect that follows the mouse pointer (optimized for desktop).
- **Reveal Animations**: Scroll-based entrance animations for sections and individual elements.
- **Responsive Navigation**: Smart, hide-on-scroll navbar with a full-screen mobile overlay.
- **Glassmorphism Design**: Modern UI elements using backdrop filters and subtle gradients.
- **Background Effects**: Floating animated orbs and a halftone dot overlay for depth.
- **Performance Optimized**: Uses `IntersectionObserver` for animations and `localStorage` to handle repetitive visit logic.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure with SEO best practices.
- **CSS3**: Vanilla CSS with custom properties (CSS variables), Bootstrap 5 for grid/layout utilities, and complex keyframe animations.
- **JavaScript (ES6+)**: Custom logic for UI interactions, smooth scroll, and animation triggers.
- **Fonts**: Outfit, Inter, and JetBrains Mono (via Google Fonts).
- **Icons**: Font Awesome 6.

## ⚙️ Getting Started

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```

2. **Open with Live Server**:
   The project uses vanilla technologies and can be run by opening `index.html` in any modern browser. For the best development experience, use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code.

### Customization

- **Updating Info**: Modify the content in `index.html` (lines 50-170).
- **Changing Colors**: Edit the CSS variables in `:root` inside `style.css`.
- **Replacing CV**: Swap `Abdelrahman osama CV.pdf` with your updated file.
